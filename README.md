# Student Planner App - Mobile Application Project

##  Project Overview
A React Native mobile application for students to manage academic activities including subject tracking, attendance monitoring, and task management.

##  Requirements Met
| Requirement | Implementation |

| 8+ Screens | Login, Register, Home, Subjects, Attendance, Settings, SubjectDetails, TaskDetails |
| GET, POST, PUT API | MockAPI integration for subject/task management |
| Stack + Tab Navigation | React Navigation as per Lab 3 instructions |
| UI Framework | React Native Paper with ready-made controls |
| Device Sensors | Camera (QR scanning) + Accelerometer (shake detection) |
| Remote Repository | GitHub with 5+ commits |

## How to Run
1. Clone: `git clone https://github.com/PanasheKeith14/student-planner-app.git`
2. Install: `npm install`
3. Start: `npx expo start`
4. Scan QR with Expo Go app

##  Technologies
- React Native with Expo
- TypeScript
- React Navigation (Stack + Bottom Tabs)
- React Native Paper (UI Components)
- Expo Camera & Sensors
- MockAPI (REST API)
